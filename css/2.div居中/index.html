水平居中

行内元素: text-align:center

块级元素：margin: 0 auto
不定宽：absolute + left:50% + translate(-50%, 0)
定宽：absolute + left:50% + margin-left: -width/2
定宽：absolute + left:0; right:0 + margin:0 auto
flex布局：justify-content:center

垂直居中

行内元素: line-height = 父元素高度

行内块级元素：父元素after伪元素+子元素均设置 vertical:middle; display: inline;
元素不定高：父元素table-cell + vertical-align
元素不定高：absolute+top+translate
元素定高：absolute+ top+ margin-top:-height/2
元素定高：absolute+ top:0;bottom:0;+ margin:auto 0;
flex布局：align-items：center;

水平垂直居中

元素定高：absolute+ top:50%; left:50%;+ margin:-height/2 0 0 -width/2;
元素定高：absolute+ top:0; left:0; bottom:0; right:0+ margin:auto;
元素不定高：absolute+ top:50%; left:50%;+ transform:translate(-50%,-50%);
flex布局：align-items：center;justify-content:center
